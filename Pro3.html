Condiciones

If / Else / Switch


Definición


condiciones u operadores relacionales permiten escribir una o varias
expresiones donde se pueden comparar para una posterior ejecución o
resultado.





Condición If


Un if se utiliza para evaluar una expresión condicional, si se cumple
la condición (verdadera), ejecutará un bloque de código






Condición If / Else


Con esta condición podemos evaluar si NO se cumple la condición.
Se hace con la palabra reservada else, nos permite ejecutar otras
instrucciones




Estructura





Switch


La estructura de selección múltiple se compone de múltiples selecciones,
donde se evalúa una expresión y se compara con los casos necesarios
y posteriormente arrojar un resultado, en caso de no encontrar ningún
caso se ejecuta las sentenciasdel argumento default
