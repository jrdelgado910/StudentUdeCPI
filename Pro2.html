Variables / Constantes

Definición y declaración


¿Qué es una variable?


Una variable es un nombre para una zona de memoria que almacena un
valor de algún tipo, esta información es asignada automáticamente
por el compilador a una dirección de la memoria



Tipos de variables


  • Int

    • Almacena típicamente cualquier número entero



  • Float

    • Almacena numeros decimales



  • Double

    • Almacena numeros decimales con una mayor cantidad de digitos



  • Char

    • Almacena 2 valores - true or false



  • String

    • Se almacena en un arreglo valores alfanumericos





    Declaracion

    Usamos la sintaxis según el lenguaje en este caso c++, indicamos
    el tipo de variable, con su nombre unico y el valor que este almacena

    ¿Qué es una constante?


    Las constantes tienen un valor fijo durante toda la ejecución del programa,
    es decir, este valor no cambia ni puede ser cambiado a lo largo de la
    ejecución de nuestro programa.
