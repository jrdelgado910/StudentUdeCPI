Potencias


Introducción


Las potencias son una operación matemática entre dos términos denominados: base a a y exponente n . Se escribe An y se lee normalmente como «a elevado a la n». Hay algunos números exponentes especiales como el 2, que se lee al cuadrado, y el 3, que se lee al cubo.

Se llama potencia a una expresión de la forma a n donde a es la base y n n es el exponente. Su definición varía según el conjunto numérico al que pertenezca el exponente. La base se multiplica por sí misma las veces indicadas por el exponente menos 1. Así, para elevar al cuadrado se multiplica una vez, y para elevar al cubo, dos veces.

La potenciaciación es una operación que consiste en multiplicar por si mismo un número principal llamado base, tantas veces como lo indique otro número que se llama exponente.



Potencia de una potencia


La potencia de una potencia es otra potencia con la misma base y cuyo exponente es el producto de los exponentes.

(am)n = am · n
