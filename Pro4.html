Repeticiones


El lenguaje C++ ofrece distintos tipos de sentencias de repetición para poder
hacer que se repita un conjunto de instrucciones, estas instrucciones se les
denomina bucles





Ciclo while


Este ciclo consiste en repetir una sentencia N veces, el ciclo se repite
hasta que la condición sea falsa y continuar con el flujo de instrucciones






Estructura





Ciclo Do while


Este ciclo es similar al While con la diferencia en que la condición se
evalúa al finalizar el ciclo, donde se ejecuta primero la sentencia y luego
se evalúa N veces






Estructura





Ciclo For


Este ciclo repetitivo se caracteriza por la facilidad de especificar el
incremento del bucle, esto quiere decir que por N repeticiones va
aumentando a medida que se ejecuta las sentencias
